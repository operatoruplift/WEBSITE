# Operator Uplift Desktop App

This directory contains configuration for building the Operator Uplift desktop app using Tauri.

## Prerequisites

- [Rust](https://rustup.rs/) (latest stable)
- [Tauri CLI](https://v2.tauri.app/start/prerequisites/)
- Node.js 18+

## Quick Start

```bash
# From the repo root
npm install
npm run build:desktop   # builds static export
cd desktop
cargo tauri dev         # runs dev mode
cargo tauri build       # creates distributable
```

## How It Works

The desktop app wraps the Next.js static export (`out/` directory) in a native Tauri window. This gives you:

- Native window with system tray
- Auto-updater
- Local file system access
- ~5MB installer size
- macOS, Windows, Linux builds

## Architecture

```
website/              ← This repo (Next.js web app)
├── out/              ← Static export (generated by `next build`)
├── desktop/          ← Tauri configuration
│   ├── src-tauri/    ← Rust backend
│   ├── tauri.conf.json
│   └── README.md
```

The Tauri app loads `out/index.html` as its webview content. All routing, state, and API calls work exactly as they do in the browser.

## Environment Variables

The desktop app uses the same env vars as the web app. Create a `.env` file in the repo root:

```
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```
